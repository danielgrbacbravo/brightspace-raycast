# Brightspace Raycast Extension

Raycast extension for viewing Brightspace courses, content, assignments, quizzes, and grades.

## Development

Work from the extension package directory:

```sh
cd brightspace
npm install
npm run dev
```

Useful commands:

```sh
npm run build
npm run lint
npm run fix-lint
```

`npm run dev` starts Raycast's extension development mode. `npm run build` compiles the extension into Raycast's local extension output directory.
